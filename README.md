# README

## What it is

An extensible C++ code for the study of ultra-cold atomic systems in the context of Gross-Pitaevskii theory. 

## Prerequisites and platforms

By now, only tested on Linux machines. 
To use it, first, you need Intel oneAPI, then also cmake. It is not tested with versions of Parallel studio previous to 
Intel oneAPI.
