\hypertarget{example-1_Title-1}{}\doxysection{A three-\/dimensional Bose gas in a harmonic trap.}\label{example-1_Title-1}
\hypertarget{example-1_Introduction-1}{}\doxysubsection{Introduction}\label{example-1_Introduction-1}
In the first example, we are going to use Ultra\+Cold to study the ground state and a simple dynamics of a three-\/dimensional, harmonically trapped Bose gas of Rubidium atoms, using the solver class {\ttfamily \mbox{\hyperlink{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver}{Ultra\+Cold\+::\+GPSolvers\+::\+GPSolver}}}, or, better, {\bfseries{extending}} this class for customizing the runtime behavior of real-\/time simulations. All the solver classes delivered with Ultra\+Cold can, in fact, be partially extended according to possible different needs, in particular for customizing the output of dynamic simulations or adding time-\/dependent terms to the Hamiltonian of the system to simulate, e.\+g., a ramp in the scattering length.

We will, in particular, solve the Gross-\/\+Pitaevskii equation on a three-\/dimensional, homogeneous mesh, calculating first the ground state of the system in the presence of an isotropic harmonic trap, using the member function {\ttfamily run\+\_\+gradient\+\_\+descent}, and then running a real-\/time dynamic simulation of an experiment for the measurement of the frequency of the so-\/called {\itshape breathing mode}, using the member function {\ttfamily run\+\_\+operator\+\_\+splitting}. The frequency of this collective oscillation can be calculated analytically in the Thomas-\/\+Fermi approximation, and is equal to \href{https://journals.aps.org/prl/abstract/10.1103/PhysRevLett.77.2360}{\texttt{ $ \omega_{breathing}=\sqrt{5}\omega_{ho}$}}. The breathing oscillation can be excited, for example, via a sudden isotropic shrinking of the harmonic trap. For the sake of illustration, however, we will use another procedure, namely a linear ramp in the scattering length, with a fixed time duration. This procedure, being isotropic in space, will still excite only the breathing mode we are interested in. It will also give us the opportunity to show how to derive a class from the base class {\ttfamily  GPSolver } in order to introduce a time-\/dependent term in the Hamiltonian.

So, let\textquotesingle{}s start from the Gross-\/\+Pitaevskii equation in three space dimensions, in the presence of an external harmonic potential

\[
    i\hbar\frac{\partial \psi(x,y,z,t)}{\partial t} =
    \left[
        \frac{-\hbar^2\nabla^2}{2m}+\frac{1}{2}m(\omega^2_x x^2+\omega^2_y y^2 +\omega_z^2 z^2)
        + \frac{4\pi a\hbar^2}{m}|\psi(x,y,z,t)|^2
    \right]\psi(x,y,z,t)
\]

Measuring frequencies in units of the average harmonic frequency $ \omega_{ho} = (\omega_x\omega_y\omega_z)^{\frac{1}{3}} $, lengths in units of the harmonic oscillator length $ a_{ho} = \sqrt{\frac{\hbar}{m\omega_{ho}}} $, and times in units of $ \omega_{ho}^{-1} $, the equation can be recast in a-\/dimensional form as

\[
    i\frac{\partial \psi(x,y,z,t)}{\partial t} =
    \left[
        \frac{-\nabla^2}{2}+\frac{1}{2}(\omega_x^2 x^2+\omega^2_y y^2+\omega_z^2 z^2)
        + 4\pi a  |\psi(x,y,z,t)|^2
    \right]\psi(x,y,z,t)
\]

We will now see how to solve this equation for our needs using tools provided by Ultra\+Cold.\hypertarget{example-1_What-1}{}\doxysubsection{Program description}\label{example-1_What-1}
