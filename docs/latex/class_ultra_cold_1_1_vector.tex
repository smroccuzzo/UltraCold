\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Ultra\+Cold\+::\+Vector$<$ T $>$}
A class that represents arrays of numerical elements. 

\begin{DoxyAuthor}{Author}
Santo Maria Roccuzzo (\href{mailto:santom.roccuzzo@gmail.com}{\texttt{ santom.\+roccuzzo@gmail.\+com}})
\end{DoxyAuthor}
The purpose of this class is to provide a useful way to store arrays of numerical elements in contiguous memory spaces and to use optimized mathematical operations acting on them. ~\newline
These include\+:


\begin{DoxyItemize}
\item Optimized memory allocation through Intel\textquotesingle{}s MKL memry functions,
\item Fourier transforms (both forward and backward),
\item Fortran-\/style access to the array elements
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Explicit instantiations are provided only for types types {\ttfamily double} and {\ttfamily std\+::complex$<$double$>$}, which is equivalent to the C complex type defined in {\ttfamily complex.\+h}. 
\end{DoxyNote}
