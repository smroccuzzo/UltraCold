\doxysubsection{Detailed Description}
A class to output a data \mbox{\hyperlink{class_ultra_cold_1_1_vector}{Vector}} in real space. 

\begin{DoxyAuthor}{Author}
Santo Maria Roccuzzo (\href{mailto:santom.roccuzzo@gmail.com}{\texttt{ santom.\+roccuzzo@gmail.\+com}})
\end{DoxyAuthor}
This class allows to write output datafiles in different formats, ready for plots with common visualization tools such as gnuplot, matplotlib, Paraview or Visit. The currently supported data output formats are\+:


\begin{DoxyItemize}
\item .csv
\item .vtk
\end{DoxyItemize}
