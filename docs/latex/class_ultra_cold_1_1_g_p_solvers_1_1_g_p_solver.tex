\doxysubsection{Detailed Description}
Class to solve the Gross-\/\+Pitaevskii equation. 

\begin{DoxyAuthor}{Author}
Santo Maria Roccuzzo (\href{mailto:santom.roccuzzo@gmail.com}{\texttt{ santom.\+roccuzzo@gmail.\+com}})
\end{DoxyAuthor}
This class allows to solve the Gross-\/\+Pitaevskii equation

\[
 i\hbar\frac{\partial \psi}{\partial t} = \left[ \frac{-\hbar^2\nabla^2}{2m}+V_{ext}({\bf r})+g|\psi|^2
\right]\psi
\]

both for ground-\/state configurations as well as for the dynamics, on a cartesian mesh with periodic boundary conditions. The basic usage of the class is as follows\+:

In the constructor, you need to provide the (cartesian) axis along which you want to solve the equation (of course, 1, 2 or 3), the initial wave function (all the methods are iterative) and the external potential. All of these must be properly initialized before passing them to the solver class. ~\newline
For ground-\/state calculations, it is possible to generate some output at each step of the gradient-\/descent iterations. By default, the \mbox{\hyperlink{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver_a01be3bb8b4fb4751bc50903a5db9b015}{GPSolver\+::run\+\_\+gradient\+\_\+descent()}} just writes, in the standard output, the current iteration number, chemical potential and norm of the residual. This default behavior can however be customized by overriding the \mbox{\hyperlink{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver_a622bd85894d716dcabe1778d08e6dad3}{GPSolver\+::write\+\_\+gradient\+\_\+descent\+\_\+output()}} member function. ~\newline
A similar behavior is the default also for dynamics calculations, i.\+e. for the member function \mbox{\hyperlink{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver_a9a741b5b2f3f801f2ffd1098393ace78}{GPSolver\+::run\+\_\+operator\+\_\+splitting()}}. In this case, the default is just to write the current time step and current time. For examples of usage and customization via overloading, see the example 1 in the {\ttfamily examples} folder. \begin{DoxyNote}{Note}
Several member functions of this class take advantage of \href{https://www.openmp.org/}{\texttt{ Open\+MP }} parallelization. For optimal performance, be sure to run 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$ export OMP\_NUM\_THREADS=<number of physical cores on the machine used>}

\end{DoxyCode}
 on the shell before the program execution. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
The Gross-\/\+Pitaevskii equation is solved in its a-\/dimensional form. In the case of a harmonic potential in one space dimension it has the form
\end{DoxyWarning}
\[
 i\frac{\partial \psi}{\partial t} = \left[ -\frac{1}{2}\frac{\partial^2}{\partial x^2}+\frac{1}{2}x^2
 +4\pi a|\psi|^2 \right]\psi
\] where $ a $ is the scattering length, which here must be given in harmonic units. Be sure to provide the axis, initial wave function, and external potential initialized properly. See the examples in the examples folder.

The Gross-\/\+Pitaevskii equation allows to obtain information on the ground-\/state properties of an ultra-\/cold bosonic system by searching for stationary solutions of the form $ \psi({\bf r},t)=\psi_0({\bf r})e^{-i\mu t/\hbar} $, obtaining the stationary Gross-\/\+Pitaevskii equation \[
\mu \psi_0 = \left[ \frac{-\hbar^2\nabla^2}{2m}+V_{ext}({\bf r})+ g|\psi_0|^2 \right]\psi_0
\] where $ g $ is related to the s-\/wave scattering length $ a $ by $ g=\frac{4\pi\hbar^2 a}{2m} $. Solving this for the smallest eigenvalue $ \mu $, which represents the chemical potential, gives access to the ground-\/state configuration of the system. ~\newline
In order to solve this equation, one can notice that it can be obtained from a constrained minimization formulation of the problem, in particular by requiring that the ground-\/state order parameter of the system is the exact minimizer of the mean-\/field energy functional \[
E[\psi] = \int d{\bf r} \left[ \psi^*({\bf r})\left(\frac{-\hbar^2\nabla^2}{2m}+
V_{ext}({\bf r})\right)\psi({\bf r}) \right] +\frac{g}{2}\int d{\bf r} |\psi({\bf r})|^4
\] under the constraint of a fixed number of particles $ \int d{\bf r}|\psi({\bf r})|^2=N $. This allows also to introduce the chemical potential $ \mu $ as the Lagrange multiplier fixing the total number of particles. One can hence find the ground state order parameter and chemical potential using, for example, a \href{https://en.wikipedia.org/wiki/Gradient_descent}{\texttt{ gradient descent }} method, which is the one implemented in this function. ~\newline
The idea is to start from a guess solution $\psi_0$ and generate a sequence of iterates $ \{\psi_n\}_{\{n=0,...,\infty\}} $ that terminates when one is sufficiently confident to have reached a (hopefully global) minimizer of the mean-\/field energy functional with good accuracy. In particular, a good stopping criterion consists in fixing a tolerance threshold $\epsilon$ (which, for this function, is a user-\/defined input parameter) for the norm of the residual, i.\+e. $ || \hat{H}\psi_n − \mu_n \psi_n ||^2 \leq \epsilon $, where $\hat{H}$ is the mean-\/field Hamiltonian of the system and the estimate $ \mu_n $ of the chemical potential $ \mu $ at iteration $ n $ can be calculated as $ \mu_n = \langle \psi_n | \hat{H} | \psi_n \rangle /  \langle \psi_n | \psi_n \rangle $. ~\newline
In deciding how to move from one iterate $ \psi_n $ to the next $ \psi_{n+1} $, line search algorithms like the gradient descent method use information about the functional $ E[\psi] $ at $ \psi_n $, and possibly also from earlier iterates $ \psi_0, \psi_1, \dots, \psi_{n-1} $. The update criterion should be that the energy functional is smaller in $ \psi_{n+1} $ then in $ \psi_n $. One thus generates a sequence $ \psi_{n+1} = \psi_n + \alpha \chi_n $ such that $ E[\psi_{n+1}]
< E[\psi_n] < \dots < E[ \psi_0 ] $ until the stopping criterion is satisfied. The update ”direction” $ \chi_n $ must thus be chosen to be a descent direction, i.\+e. a direction along which the functional $ E[\psi] $ decreases. The step-\/length $ \alpha $ should instead be (ideally) chosen in such a way that the decrease in the energy functional is, at each iteration step, the maximum possible. Since this is not, in general, an easy task, in this function we accept the compromise to choose the step length $ \alpha $ empirically as an input parameter at the beginning of the iteration procedure. This also implies that the user may need to run the function a few times before finding an optimal value for $ \alpha $. ~\newline
Coming back to the choice of $ \chi_n $, the gradient descent method consists in choosing such descent direction as the opposite of the gradient of the functional $ E[\psi] $ calculated in $ \psi_n $ . So, in this case, the descent direction is (minus) the functional derivative of the energy functional with respect to $ \psi^* $ evaluated at $ \psi_n $ , i.\+e. \[

\chi_n = - \frac{\delta E[\psi_n]}{\delta \psi^*} =
- \left[ \frac{-\hbar^2\nabla^2}{2m}+V_{ext}({\bf r})+g|\psi_n|^2 \right] \psi_n

\] The ground state wave function obtained from this algorithm is normalized in such a way to preserve the initial norm, i.\+e. the norm of the initial wave function provided. Such normalization condition, fixing the $ L^2 $ norm of the ground-\/state wave-\/function $ \psi $ should be included in the iteration procedure by introducing a Lagrange multiplier and minimizing the corresponding lagrangian functional. In practice, it is much cheaper to normalize ”by hand” each $ \psi_n $ obtained via the gradient descent iteration, by fixing \[
 \psi_{n+1}^{(1)} = \psi_n + \alpha \chi_n
\] \[
\psi_{n+1} = \sqrt{\frac{N}{\int d{\bf r}|\psi_{n+1}^{(1)}|^2}}\psi_{n+1}^{(1)}
\] where $ N $ is the initial norm. Notice that, physically, this represents the total number of particles in the system, and as such must be an integer. Hence, the initial wave function must be properly normalized before the gradient-\/descent iterations start. ~\newline
Finally, this function employs an acceleration algorithm, known as the \href{https://www.worldscientific.com/doi/abs/10.1142/S0219199700000025}{\texttt{ heavy ball method }}, in order to speed up the convergence of the sequence $ {\psi_n} $. The method consists in adding a ”momentum” term into the gradient-\/descent iterations, in order to make larger steps if the descent direction does not change very much, and smaller steps if it changes a lot. In practice, what one does is just to modify the gradient-\/descent expression in $ \psi_{n+1} = \psi_n + \alpha \chi_n + \beta(\psi_n-\psi_{n-1}) $ where, again, $ \beta $ is a parameter chosen empirically and given to this function as input. \mbox{\Hypertarget{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver_a9a741b5b2f3f801f2ffd1098393ace78}\label{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver_a9a741b5b2f3f801f2ffd1098393ace78}}

Solve the Gross-\/\+Pitaevskii equation using simple operator splitting. 


\begin{DoxyParams}{Parameters}

\hline

This member function solves the (a-\/dimensional) time-\/dependent Gross-\/\+Pitaevskii equation \[
i \frac{\partial \psi}{\partial t} = \left[ \frac{-\nabla^2}{2}+V_{ext}({\bf r})+g|\psi|^2 \right]\psi
\] using the classic operator splitting technique.~\newline
The general idea of operator-\/splitting methods is to consider an initial value problem \[
y\textnormal{\textquotesingle} = \text{A} y + \text{B} y
\] where $ A $ and $ B $ are differential operators, and solve the equation considering the actions of the two operators separately. There is a vast literature on operator-\/splitting approaches for the solution of differential equations, and different methods with a higher or lower level of accuracy. In the case of the Gross-\/\+Pitaevskii equation, things are even more simplified by the fact that part of the method implies steps that can be solved {\itshape exactly}. ~\newline
After choosing a time-\/step $ \Delta t $, the operator spltting scheme consists in the following steps \[
\begin{align}
& \text{For}   \quad n=0,1,\dots,\text{number_of_time_steps} \nonumber \\
& \text{Step 1: solve} \quad y\textnormal{\textquotesingle}_1 = \text{A}y_1 \quad \text{in} \quad [t_n,t_n+\Delta t]
\quad \text{for} \quad y_1(0)=y(t_n) \nonumber \\
& \text{Step 2: solve} \quad y\textnormal{\textquotesingle}_2 = \text{B}y_2 \quad \text{in} \quad [t_n,t_n+\Delta t]
\quad \text{for} \quad y_2(0)=y_1(t_n+\Delta t) \nonumber \\
& \text{Set} \quad y(t_{n+1}) = y_2(t_n+\Delta t) \nonumber \\
\end{align}
\] The steps imply the solution of an ordinary differential equation. In the case of the Gross-\/\+Pitaevskii equation, a good choice of the operators $ A $ and $ B $ is the following \[
\begin{align}
 & A = V_{ext} + g |\psi(t)|^2 \nonumber \\
 & B = \frac{-\nabla^2}{2} \nonumber \\
\end{align}
\]

In this case, step 1, although involve the solution of a non-\/linear differential equation, can be solved {\bfseries{analytically}}. In fact, it is easy to show that the equation \[
 i\frac{d}{dt}\psi(t) = V_{ext} + g |\psi(t)|^2
\] preserves the norm $ \psi $ in time, and hence an explicit solution of this equation is \[
\psi(t+\Delta t) = e^{-i\Delta t\left( V_{ext} + g |\psi(t)|^2 \right)}\psi(t)
\] Moreover, also the second step of the method can be solved analytically, provided that one imposes {\itshape periodic boundary conditions}. In fact, given the equation \[
i\frac{d}{dt}\psi(t) = \frac{-\nabla^2}{2}\psi(t)
\] and taking the Fourier transform (in space) at both sides, one finds \[
i\frac{d}{dt}\tilde{\psi}(t) = \frac{k^2}{2}\tilde{\psi}(t)
\] which can again be solved exactly as \[
\tilde{\psi}(t+\Delta t) = e^{-i\Delta t\frac{k^2}{2}}\tilde{\psi}(t)
\] Finally, an inverse Fourier transform allows to recover the solution back in real space, ready to take another time step. ~\newline
To summarize, this member function solves the Gross-\/\+Pitaevskii equation using classic operator splitting via the following steps
\begin{DoxyItemize}
\item Step 1) Set $ \psi(t_n + \Delta t) =
 e^{-i\Delta t\left( V_{ext} + g |\psi(t_n)|^2 \right)}\psi(t_n) $;
\item Step 2)
\begin{DoxyItemize}
\item 2.\+1) Take the Fourier transform of $ \psi(t_n + \Delta t) $, and call it $ \tilde{\psi}(t_n) $;
\item 2.\+2) Set $ \tilde{\psi}(t_n+\Delta t) = e^{-i\Delta t\frac{k^2}{2}}\tilde{\psi}(t_n) $
\item 2.\+3) Take the inverse Fourier transform of $ \tilde{\psi}(t_n+\Delta t) $, and obtain $ \psi(t_n+\Delta t) $
\end{DoxyItemize}
\end{DoxyItemize}

Notice that all these steps can be computed {\itshape locally}, meaning that we don\textquotesingle{}t need additional vectors to store intermediate values of $ \psi $. \mbox{\Hypertarget{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver_ac7e742ac1e567454d30a69ad50df24d2}\label{class_ultra_cold_1_1_g_p_solvers_1_1_g_p_solver_ac7e742ac1e567454d30a69ad50df24d2}}
