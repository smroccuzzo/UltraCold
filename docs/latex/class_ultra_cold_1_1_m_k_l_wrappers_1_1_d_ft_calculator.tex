\doxysubsection{Detailed Description}
Class to calculate Fourier transforms using Intel\textquotesingle{}s MKL DFT functions. 

\begin{DoxyAuthor}{Author}
Santo Maria Roccuzzo (\href{mailto:santom.roccuzzo@gmail.com}{\texttt{ santom.\+roccuzzo@gmail.\+com}})
\end{DoxyAuthor}
The purpose of this class is to provide a simple way for calculating Fourier transforms using Intel\textquotesingle{}s MKL Discrete Fourier Transform functions. ~\newline
The class is used as follows

Here, {\ttfamily forward\+\_\+domain\+\_\+vector} is the real-\/space vector, while {\ttfamily backward\+\_\+domain\+\_\+vector} is the Fourier-\/space vector. \begin{DoxyNote}{Note}
The two Vectors must be different (out-\/of-\/place transform). Moreover, for performance and portability reasons, a {\ttfamily \mbox{\hyperlink{class_ultra_cold_1_1_m_k_l_wrappers_1_1_d_ft_calculator}{DFt\+Calculator}}} object must be associated with {\bfseries{each}} forward-\/backward pair of vectors you want to use. 
\end{DoxyNote}
