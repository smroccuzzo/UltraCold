Detailed Description

Namespace containing GPU-accelerated solvers for various flavors of Gross-Pitaevskii equations.

Author
Santo Maria Roccuzzo (santo.nosp@m.m.ro.nosp@m.ccuzz.nosp@m.o@gm.nosp@m.ail.c.nosp@m.om)

This namespace contains several GPU-accelerated solver classes for various flavors of Gross-Pitaevskii equations. Such solvers are implemented in CUDA, and all the computationally intensive tasks are delegated to a GPU. In order to use such classes, it is necessary of course to have an NVIDIA GPU available, as well as CUDA installed.
